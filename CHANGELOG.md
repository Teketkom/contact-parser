## [2.0.0] — 2026-03-04

### Добавлено
- **Документация проекта** — полный комплект документации создан:
  - `docs/SYSTEM_PROMPT.md` — системный промпт для AI-агента (ПАРСЕР.СП.001 v2.0)
  - `docs/ARCHITECTURE.md` — детальная архитектура с ASCII-диаграммами (ПАРСЕР.АРХ.001)
  - `docs/API.md` — документация REST API с примерами curl (ПАРСЕР.АПИ.001)
  - `docs/DEPLOYMENT.md` — инструкции по развёртыванию (Docker, bare metal) (ПАРСЕР.ДЕП.001)
  - `docs/TESTING.md` — стратегия тестирования и примеры тестов (ПАРСЕР.ТСТ.001)
  - `docs/DOCUMENT_PIPELINE.md` — конвейер документов проекта (ПАРСЕР.ДОК.001)
  - `README.md` — полное пользовательское руководство
  - `CONTRIBUTING.md` — руководство по участию в разработке
  - `CHANGELOG.md` — данный файл
  - `LICENSE` — MIT License
- **Структура репозитория** — создана полная директорная структура проекта
- **Реестр документов** — сформирован и задокументирован реестр из 14 проектных документов
- **Метрики качества** — определены целевые значения: Precision ≥ 80%, Recall ≥ 70%
- **Стандарты разработки** — описаны coding standards, git flow, conventional commits
- **CI/CD pipeline** — описан конвейер: lint → test → build → security → release
